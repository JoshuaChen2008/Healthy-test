> 填的时候特别留意：
>
> - 有没有让你选**单位**（cm/kg vs 英尺/磅）？
> - 有没有让你填**邮箱**或注册？
> - "目标"是**只能选一个**还是**能选多个**？
> - 有没有问我们完全没想到的东西（睡眠、饮食习惯、身体部位…）？

## B. 数据保存时机（对应"分步保存"）

- 每答完一步，页面是"立刻走到下一步"还是有"保存/下一步"按钮？
- 中途刷新页面，进度还在吗？（这决定"进度恢复"有多重要）

## C. 付费墙前后差异（对应"脱敏 vs 完整"）—— 最重要

| 项目 | 付费前（非会员）看到 | 付费后（会员）看到 |
|------|---------------------|---------------------|
| BMI / 身体指标 |  | |
| 目标日期 / 预测曲线 | | |
| 计划 / 方案内容 | 只能看见一个粗略的曲线图，告诉你跟着计划可以达到什么效果 | 详细的计划，通过邮件发送 |
| 其他 | | |

> 结论：哪些字段是"藏起来诱导付费"的 → 这些就是我们的**被保护字段**。
